# Knowledge Atlas

Knowledge Atlas is a full-stack TypeScript learning project for understanding Knowledge Graphs, Vector Databases, and Retrieval-Augmented Generation, also known as RAG.

The goal is to build a practical AI research assistant that can ingest documents, extract concepts, store relationships in a knowledge graph, store semantic vectors in a vector database, and answer questions using retrieved evidence.

## What This Project Teaches

- Full-stack TypeScript development
- Backend API design
- Frontend development
- Knowledge Graphs with Neo4j
- Vector search with Qdrant
- RAG pipeline design
- Testing
- CI/CD
- Deployment

## Planned Architecture

```text
Frontend App
  -> Backend API
  -> Qdrant Vector Database
  -> Neo4j Knowledge Graph
  -> Local or hosted LLM
```
